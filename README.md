# 3Days Programming Bootcamp
多くのWebサービスで使われる共通の仕組みを、わずか3日間でマスターするスペシャルレクチャー。Webサービスが動く原理からフロントエンド技術、バックエンド技術までをGoodfindの現役エンジニアが網羅的・実践的に教えます。
